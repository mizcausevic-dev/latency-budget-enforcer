# Security Policy

## Supported Use

This repository is a portfolio project that demonstrates latency governance and enforcement patterns. It should not be treated as a production control without environment-specific hardening, telemetry integration, and security review.

## Reporting a Vulnerability

If you discover a vulnerability in this repository, please report it privately through GitHub security reporting or by contacting the maintainer through the channels listed on [kineticgain.com](https://kineticgain.com/).

Please include:

- affected file or component
- reproduction steps
- expected impact
- suggested remediation if available
